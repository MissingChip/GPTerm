# GPTerm

Use LLMs in the terminal.

Just install
```
pip install .
```

Set the API key
(how you do this depends on your shell)
```
OPENAI_API_KEY=API_KEY
```

And run
```
chat
```
or
```
gpterm
```

It's also recommended to copy `.env.example` to `.env` and replace any values
(particularly OPENAI_API_KEY) with your own
